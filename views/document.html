Get all Users

GET

Request: /api/users

Response: return all users

Get User by ID

GET

Request: /api/users/id/:userID

Response: return specific user

Get User by name

GET

Request: /api/users/name/:userName

Response: return specific user's correlateduser name

Get all articles

GET

Request: /api/articles

Response: return all articles

Get Article by ID

GET

Request: /api/articles/id/:articleID

Response: return specific article

Get top groups

GET

Request: /api/groups/date/:date/range/:range

Response: [{"group_id":0,"group_count":656}]

Get User's Group

GET

Request: /api/groups/userID/:userID

Response: return the groupID of the User

Get Group's Feature

GET

Request: /api/features/group/:groupID

Response: return the featurelist of the group

Get Visual Data

GET

Request: /api/visual/date/:date/range/:day_range

Response: return the nodes and links of the group result of that date and day_range
